Introduction to C++

C++ is a powerful, general-purpose programming language developed by Bjarne Stroustrup in the early 1980s. It is an extension of the C programming language and supports both procedural and object-oriented programming.

C++ is widely used in software development, game development, system programming, embedded systems, and competitive programming. It provides high performance and fine control over system resources.

Due to its efficiency and flexibility, C++ is one of the most popular programming languages taught in BCS (Bachelor of Computer Science) courses.

C++ supports multiple programming paradigms such as: Procedural programming Object-oriented programming Generic programming Because of this flexibility, C++ is widely used in both academic and industrial applications.


Features of C++

  • Object-oriented programming (OOP)
  • Procedural programming
  • Generic programming
  • High performance and efficiency
  • Support for low-level memory manipulation

    • Application of C++

    C++ is used in a wide range of applications, including:

    • System software development
    • Game development
    • High-performance computing
    • Embedded systems programming
    • Real-time systems
    • Graphics and multimedia applications


    Data Types

    A data type specifies the type of data a variable can store in a program. It tells the compiler how much memory is needed and what kind of value can be stored.


    Types of Data Types

    There are two main Types of Data Types in c++

    • Primitive Data Types
    • Derived Data Types

      • Primitive Data Types

      Primitive data types are the basic data types provided by the C++ language. They are also known as built-in data types.

      • int
      • float
      • double
      • char
      • bool

      Derived Data Types

      Derived data types are created from primitive data types using various operations and declarations.

      • arrays
      • pointers
      • references
      • functions

      Variables

      A variable is a named memory location used to store data in a program. Variables are declared with a specific data type and can be assigned values during runtime.


      Rules for Naming Variables

      • Variables must start with a letter or an underscore.
      • Variables can contain letters, digits, and underscores.
      • Variables are case-sensitive.
      • Variables cannot be reserved keywords.

            control statements

            What are control statements?

            Control statements are statements that control the flow of execution of a program based on conditions, loops, or jumps.


            Types of Control Statements

            • Selection statements (if, if-else, switch)
            • Loop statements (for, while, do-while)
            • Jump statements (break, continue, return)

            What are control statements ?

            Selection statements evaluate a condition (true or false) and select one path of execution from multiple possible paths.


            Types of Control Statements

            1. if statement: Executes a block of code if a specified condition is true.
            2. ifâ€“else statement: Executes one block of code if a condition is true, and another block if it's false.
            3. else-if ladder: Allows multiple conditions to be checked in sequence.
            4. Nested if statement: An if statement inside another if statement.
            5. switch statement: Selects one of many code blocks to execute based on the value of an expression.

            What's next ?

            Next, we will learn about Loop Statements in C++ which help in executing a block of code multiple times based on a condition.


            What is Loop ?

            A loop is a control structure that allows you to repeat a block of code multiple times based on a specified condition.


            Types of Loops

            • for loop
            • while loop
            • do-while loop

            for Loop

            Used when the number of iterations is known in advance. 
                            Syntax:
                for (initialization; condition; increment/decrement) {
                }
                
                            example:
                for (int i = 1; i <= 5; i++) {
                         cout << i << " ";
                }

            working

            1. Initialization: Sets the starting value of the loop control variable.
            2. Condition: Evaluated before each iteration; if true, the loop body executes.
            3. Increment/Decrement: Updates the loop control variable after each iteration.

            while Loop

            Used when the number of iterations is unknown and depends on a condition. 
                                Syntax:
                    while (condition) {
                    }
                
                                example:
                    int i = 1;
                    while (i <= 5) {
                        cout << i << " ";
                        i++;
                    }

            do-while Loop

            Similar to while loop, but guarantees at least one execution of the loop body. 
                                Syntax:
                    do {
                    } while (condition);
                    
                                example:
                    int i = 1;
                    do {
                        cout << i << " ";
                        i++;
                    } while (i <= 5);

            Comparison of loops

            Loop Type Use Case Condition Check
            for loop Known iterations Before each iteration
            while loop Unknown iterations Before each iteration
            do-while loop At least one execution needed After each iteration

            Nested Loops

            A loop inside another loop is called a nested loop. The inner loop completes all its iterations for each iteration of the outer loop. 

                                example:
                    for (int i = 1; i <= 3; i++) {
                        for (int j = 1; j <= 2; j++) {
                            cout << "i: " << i << ", j: " << j << endl;
                        }
                    }

            output:- i: 1, j: 1 i: 1, j: 2 i: 2, j: 1 i: 2, j: 2 i: 3, j: 1 i: 3, j: 2


            What is loop statements ?

            Loop control statements are used to change the normal flow of a loop. They allow us to terminate a loop, skip iterations, or transfer control as required.


            Types of Loop Control Statements

            • break statement
            • continue statement
            • return statement

            break statement :-

            Used to terminate the loop immediately and transfer control to the statement following the loop. 
                                    Syntax:
                        break;
                        
                                    example:
                        for (int i = 1; i <= 10; i++) {
                            if (i == 5) {
                                break; // exit loop when i is 5
                            }
                            cout << i << " ";
                        }

            continue statement :-

            Used to skip the current iteration of the loop and proceed to the next iteration. 
                                    Syntax:
                        continue;
                        
                                    example:
                        for (int i = 1; i <= 10; i++) {
                            if (i % 2 == 0) {
                                continue; // skip even numbers
                            }
                            cout << i << " ";
                        }

            return statement :-

            Used to exit from a function and optionally return a value to the caller. 
                                    Syntax:
                        return value;
                        
                                    example:
                        int sum(int a, int b) {
                            return a + b; // return the sum of a and b
                        }

            goto statement :-

            Used to transfer control to a label within the same function. 
                                        Syntax:
                            goto label;
                            
                                        example:
                            int main() {
                                int i = 1;
                                loop:
                                    cout << i << " ";
                                    i++;
                                    if (i <= 5) {
                                        goto loop; // jump to label 'loop'
                                    }
                                return 0;
                            }

            Difference between break, continue, and return statements

            Statement Purpose Effect
            break Terminate loop Exits the loop immediately
            continue Skip iteration Skips to the next iteration of the loop
            return Exit function Returns control to the caller and optionally returns a value

            Summary of loop

            • Loops are used to repeat a block of code multiple times based on a condition.
            • Types of loops: for, while, do-while.
            • Loop control statements: break, continue, return, goto.
            • Nested loops allow loops within loops for complex iterations.

            Next, we will learn about Functions in C++, which allow us to modularize our code and reuse it efficiently.


            Functions

            A function is a block of code that performs a specific task. It allows you to break down your program into smaller, manageable parts.

            Benefits of using functions

            • Modularity: Functions help organize code into logical sections.
            • Reusability: Functions can be called multiple times without rewriting code.
            • Maintainability: Functions make it easier to update and maintain code.

            Function Declaration and Definition

            A function must be declared before it is used. The declaration specifies the function's name, return type, and parameters (if any). 

                                            Syntax:
                                return_type function_name(parameter_list);

            The function definition contains the actual code that performs the task. 

                                                Syntax:
                                    return_type function_name(parameter_list) {
                                        // function body
    }

            Example of a Function

                                                    #include 
using namespace std;
                                        // Function declaration
                                        int add(int a, int b);

                                        int main() {
                                            int num1 = 5, num2 = 10;
                                            // Function call
                                            int sum = add(num1, num2);
                                            cout << "Sum: " << sum << endl;
                                            return 0;
                                        }

                                        // Function definition
                                        int add(int a, int b) {
                                            return a + b;
                                        }

            Function Parameters and Return Types

            Functions can take parameters (inputs) and return values (outputs).

            • Parameters: Variables that hold the values passed to the function.
            • Return Type: The data type of the value that the function returns.

            Types of Functions

            • Void Functions: Do not return a value.
            • Value-returning Functions: Return a value of a specified type.
            • Recursive Functions: Call themselves to solve a problem.

            1.Void Functions: Do Not Return a Value

            Void functions do not return any value. They are used to perform a specific task. 

                            void printMessage() {
                    cout << "Hello, World!" << endl;
                }

            2.Value-returning Functions: Return a Value of a Specified Type

            Value-returning functions return a value of a specified type. 

                            int multiply(int a, int b) {
                    return a * b;
                }

            3.Recursive Functions: Call Themselves to Solve a Problem

            Recursive functions call themselves to solve a problem by breaking it down into smaller subproblems. 

                            int factorial(int n) {
                    if (n == 0) {
                        return 1;
                    }
                    return n * factorial(n - 1);
                }

            Function Overloading

            Function overloading allows multiple functions with the same name but different parameters. 

                            int add(int a, int b) {
                    return a + b;
                }

                double add(double a, double b) {
                    return a + b;
                }
