Introduction to C++

C++ is a powerful, general-purpose programming language developed by Bjarne Stroustrup in the early 1980s. It is an extension of the C programming language and supports both procedural and object-oriented programming.

C++ is widely used in software development, game development, system programming, embedded systems, and competitive programming. It provides high performance and fine control over system resources.

Due to its efficiency and flexibility, C++ is one of the most popular programming languages taught in BCS (Bachelor of Computer Science) courses.

C++ supports multiple programming paradigms such as: Procedural programming Object-oriented programming Generic programming Because of this flexibility, C++ is widely used in both academic and industrial applications.


Features of C++

  • Object-oriented programming (OOP)
  • Procedural programming
  • Generic programming
  • High performance and efficiency
  • Support for low-level memory manipulation

    • Application of C++

    C++ is used in a wide range of applications, including:

    • System software development
    • Game development
    • High-performance computing
    • Embedded systems programming
    • Real-time systems
    • Graphics and multimedia applications

    Conclusion

    C++ is a versatile and powerful programming language that plays a crucial role in various domains of computer science and software development. Its ability to combine high-level features with low-level control makes it an ideal choice for many applications.


    Data Types

    A data type specifies the type of data a variable can store in a program. It tells the compiler how much memory is needed and what kind of value can be stored.


    Types of Data Types

    There are two main Types of Data Types in c++

    • Primitive Data Types
    • Derived Data Types

      • Primitive Data Types

      Primitive data types are the basic data types provided by the C++ language. They are also known as built-in data types.

      • int
      • float
      • double
      • char
      • bool

      Derived Data Types

      Derived data types are created from primitive data types using various operations and declarations.

      • arrays
      • pointers
      • references
      • functions

      Variables

      A variable is a named memory location used to store data in a program. Variables are declared with a specific data type and can be assigned values during runtime.


      Rules for Naming Variables

      • Variables must start with a letter or an underscore.
      • Variables can contain letters, digits, and underscores.
      • Variables are case-sensitive.
      • Variables cannot be reserved keywords.

      Example of variable Declaration

      declaration without initialization:

                  int age;
            float salary;
            char grade;
            bool isPassed;

      declaration with initialization:

                  int age = 20;
            float salary = 50000.5;
            char grade = 'B';
            bool isPassed = true;

      multiple variable declarations:

                  int a, b, c;
float x = 2.5, y = 4.8;

      same value to multiple variables:

                  int x = 5, y = 5, z = 5;

      Character Variable Declaration

                  char ch = 'A';
char symbol = '#';

      Boolean Variable Declaration

                  bool status = true;
bool isLogin = false;

      string variable declaration:

                  string name = "Ayush";

      constant variable declaration

                  const float PI = 3.14;

      Example Program Using Variable Declaration

                  #include 
using namespace std;

int main() {
    int roll = 101;
    float marks = 88.5;
    char grade = 'A';
    bool pass = true;

    cout << "Roll: " << roll << endl;
    cout << "Marks: " << marks << endl;
    cout << "Grade: " << grade << endl;
    cout << "Pass: " << pass << endl;

    return 0;
}

      Operators & Expressions


      What is Operators

      An operator is a symbol that tells the compiler to perform a specific operation on one or more operands (variables, constants, or expressions).


      examples

      int a = 10, b = 5; int c = a + b;

      What is expressions

      An expression is a combination of operands and operators that produces a single value.

      example

      a + b * 2

      Types of Operators

      • Arithmetic Operators
      • Relational Operators
      • Logical Operators
      • Assignment Operators
      • Increment/Decrement Operators
      • Bitwise Operators
      • Conditional Operators
      • special Operators

      Arithmetic operators

      • Addition (+)
      • Subtraction (-)
      • Multiplication (*)
      • Division (/)
      • Modulus (%)

        • example

        int a = 10, b = 3; cout << a + b; // 13 cout << a - b; // 7 cout << a * b; // 30 cout << a / b; // 3 (integer division) cout << a % b; //1

        Relational Operators

        Used to compare two values. Result is true (1) or false (0).

        • Equal to (==)
        • Not equal to (!=)
        • Greater than (>)
        • Less than (<)
        • Greater than or equal to (>=)
        • Less than or equal to (<=)

          • Logical operators

          Used to combine multiple conditions.

          • Logical AND (&&)
          • Logical OR (||)
          • Logical NOT (!)
